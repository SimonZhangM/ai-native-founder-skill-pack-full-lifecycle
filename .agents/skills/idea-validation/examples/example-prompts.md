# Example prompts for `idea-validation`

```text
请使用 $idea-validation 处理当前问题，并按 skill 的输出格式给出阶段门、证据、风险和下一步。
```

```text
Use the idea-validation skill. First state whether the skill is applicable. If it is not applicable, explain which skill should run instead.
```

```text
请不要修改任何文件。先用 idea-validation 给出只读分析、缺失输入和建议的下一步。
```
