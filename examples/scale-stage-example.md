# Scale Stage Example

## User prompt

```text
我们已经有稳定收入，也开始卖给企业客户。但销售、支持、续约和产品反馈都还是我亲自处理。请做 scale audit。
```

## Recommended flow

1. Run `founder-bottleneck-audit`.
2. Run `scale-moat-system`.
3. Ask `reviewer` to challenge defensibility.

## Expected output

```markdown
## Scale Readiness
Decision: HOLD

## Bottleneck
Pipeline and enterprise support still depend on founder activity.

## Moat work
- Encode domain edge cases into tests/workflows.
- Build integration-depth map.
- Create customer workflow lock-in audit.
```
