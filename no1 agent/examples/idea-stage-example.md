# Idea Stage Example

## User prompt

```text
我想做一个 AI 工具，帮助中型公司的财务经理自动核对报销单和会计系统。请判断是否值得做。
```

## Recommended flow

1. Run `founder-stage-diagnosis`.
2. Run `idea-validation`.
3. Run `customer-discovery`.
4. Ask `reviewer` to challenge the conclusion.

## Expected output

```markdown
## Testable Hypothesis
Finance managers at mid-market companies spend 4+ hours per week reconciling employee expense submissions because expense tools and accounting systems do not sync cleanly.

## Gate Decision
HOLD until 5-10 target-user conversations confirm frequency, severity, current workaround, and buyer priority.
```
